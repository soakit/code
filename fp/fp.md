# 函数式编程

## js 纯函数

- 好处
  1. 可推测，方便测试
  2. 可复用，模块化
  3. 无副作用
  4. 可读性
- 特点
  - 提倡不可变数据
  - 高阶函数
    传入一个函数，处理这个函数或返回一个新的函数
    - 抽象
      - 命令式编程与声明式编程
        (如 for 循环与 forEach)
    - 缓存
      - 缓存执行结果
    - 惰性执行
      - 返回的新函数就是惰性执行的
